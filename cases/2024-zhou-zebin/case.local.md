- 「《民法典》」第158条：民事法律行为可以附条件。附生效条件的民事法律行为，自条件成就时生效。附解除条件的民事法律行为，自条件成就时失效。— 确信度 🟡（用于论证转账附有"以将来共同生活为目的"的条件）
- 「最高人民法院关于审理涉彩礼纠纷案件适用法律若干问题的规定」（2024年2月1日施行）第四条：婚约财产纠纷中，人民法院应当综合考虑彩礼的目的性特征...确定是否返还以及返还的具体比例。— 确信度 🟡
- 「《民法典》」第967-978条（合伙合同）：合伙人应当按照约定的出资方式、数额和缴付期限，履行出资义务。合伙合同终止后应当进行清算。— 确信度 🟢

### ⭐⭐⭐ 最优路径一：不当得利返还请求权

**推荐案由：不当得利纠纷**

**核心逻辑**：160.5万的转账并非合伙利润的最终分配，而是一笔"目的性给付"——给男方去投资创业，以双方将来共同生活为目的。该目的因分手而落空，构成给付目的不能实现的不当得利（目的性给付型不当得利/Zweckverfehlungskondiktion）。

**法律论证链条**：
1. 转账发生在2024.6.6，当时双方尚未分手（分手日为7.1）
2. 男方家属微信明确证实"他没有要分开的意思"
3. 转账目的是"给他去发展"——为将来共同生活
4. 男方收钱后失联，目的彻底落空
5. → 男方保有该160.5万丧失法律根据 → 构成不当得利

**要件涵摄**：
- 要件1（获利）→ ✅ 男方收到160.5万
- 要件2（受损）→ ✅ 女方支出160.5万
- 要件3（因果）→ ✅ 同一笔转账
- 要件4（无合法根据）→ ⚠️ 需重点论证转账目的与目的落空

**对方可能抗辩**：
- "这160.5万本来就是320万的一半，是我应得的利润分配" → **我方应对**：利润分配应当在合伙清算后确定，未经清算的"协议分配"不具备最终结算效力；且转账时并非以"利润分配"为目的（微信记录可证明 — 目的是去创业投资）
- "这是赠与，已交付不可撤销" → **我方应对**：非无偿赠与，有明确的对价期待（共同生活），属目的性给付

### ⭐⭐⭐ 最优路径二（可并行）：合伙合同纠纷

**推荐案由：合伙合同纠纷**

**核心逻辑**：双方自2020年起共同经营淘宝店铺，形成事实上的合伙关系。2024年7月分手即合伙终止，应当进行合伙清算。在清算完成前，男方单方占有的合伙财产应当返还至合伙体。

**突破性价值**：此路径可以将争议范围从160.5万扩大到整个合伙财产——女方净转出888万+，如果合伙清算，女方可能获得远超160.5万的回报。

**风险提示**：
- 🟡 此路径下，需要对全部资金流进行审计，举证工作量极大
- 🟡 男方的"辛马威"店铺的投入和收益也需要纳入清算范围
- 🔴 如果法院认定"分手时双方已就320万达成了分配协议"，则合伙清算的空间会被压缩

### 5.4 核心对抗策略

**对方最强抗辩**：160.5万是320万共同经营利润的一半，是最终分配，男方有权保有。

**我方攻破思路**：

1. **时间线攻击**：转账发生在6月6日，分手在7月1日。转账时双方关系尚在，不是"分手结算"行为。
2. **目的攻击**：微信记录明确显示——转钱的目的是让男方"去合作搞拼多多"，是为将来共同生活的投资行为，非利润分割。
3. **行为攻击**：如果是合理的利润分配，男方不应在收到钱后立即"失联"。这种行为本身反证了该转账并非双方达成一致的最终清算。
4. **法律攻击**：即便320万是共同经营利润，合伙财产的分配需在合伙终止后清算确定（《民法典》第978条），未经清算的单笔转账不构成最终分配。分手时"确定分给女方一半"的口头表述，不等于已完成合伙清算。
