# 保密协议审查 (NDA Review)

对保密协议进行快速结构化审查，在 15 分钟内给出风险评估和修改建议，而非逐字逐句读完后才给意见。

## 为什么NDA审查需要专门的 Skill

保密协议是最常见的法律文书之一，但大多数人签署前不会仔细审查。问题在于：NDA 中最危险的条款往往藏在定义条款和违约责任中——保密范围被无限扩大（"包括但不限于所有商业信息"）和违约金过高是最常见的陷阱。

## 审查清单

### 1. 保密范围（最重要）
- [ ] 保密信息是否有明确定义？还是用"包括但不限于"无限扩张？
- [ ] 是否约定了排除事项（公知信息、独立开发、第三方合法获取）？
- [ ] 口头信息是否也被纳入保密范围？（如果是，确认需求合理性）

### 2. 保密期限
- [ ] 期限是否合理？（一般 2-5 年为合理，"永久保密"应当警惕）
- [ ] 协议终止后保密义务是否还延续？延续多久？

### 3. 允许披露范围
- [ ] 是否明确允许向律师、审计师、关联公司等必要方披露？
- [ ] 是否约定了被法律要求披露时的处理流程？

### 4. 违约责任
- [ ] 违约金金额是否过高？（明显超过实际损失可能被认定为无效）
- [ ] 是否约定了举证责任分配？

### 5. 竞业限制（如涉及）
- [ ] 竞业限制期限是否超过 2 年？（超过 2 年的部分无效，《劳动合同法》第24条）
- [ ] 是否约定了竞业限制经济补偿金？（未约定补偿的竞业限制可能被认定无效）
- [ ] 竞业地域和行业范围是否合理？

### 6. 管辖与适用法律
- [ ] 管辖法院/仲裁机构条款
- [ ] 适用法律（涉外NDA需特别关注）

[对每个红标条款提供具体的修改措辞建议]

### 总体评价

- **签署建议**：[可签/需修改后签/建议拒绝]
- **核心关注点**：[最需要律师关注的 1-2 个问题]

## 处理规则

- **快速分诊优先**：先完成风险速览表，再对红标条款展开详细分析。律师需要的是"快速判断能不能签"，而非学术论文。
- **双向视角**：同时考虑我方作为披露方和接收方时的利益——同一条款对不同角色的影响不同。
- **竞业限制专项注意**：涉及劳动关系的竞业限制受《劳动合同法》强制约束，不能完全由双方自由约定。

## 示例

**输入：** 某公司要求员工签署的保密和竞业限制协议，保密期"永久"，竞业限制 3 年无补偿。

**输出（节选）：**

### 风险速览

| 审查项 | 状态 | 风险级别 | 说明 |
|-------|------|---------|------|
| 保密范围 | 过宽（"一切商业信息"） | 🔴 | 缺乏排除条款，建议限缩 |
| 保密期限 | 永久 | 🟡 | 商业秘密可永久，但一般商业信息不应 |
| 违约责任 | ¥100万 固定违约金 | 🔴 | 明显超出一般员工可能造成的损失 |
| 竞业限制 | 3年无补偿 | 🔴 | 超2年部分无效；无补偿可能导致整个竞业条款不可执行 |

- **签署建议**：需修改后签
- **核心关注点**：竞业限制条款因无补偿金存在被认定无效的风险
