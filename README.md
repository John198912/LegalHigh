# LegalHigh

A Python-based application for legal document processing and management.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```python
from legalhigh import LegalHigh

app = LegalHigh()
app.run()
```

## Development

```bash
# Install dependencies
pip install -r requirements.txt

# Run tests
python -m pytest tests/
```
